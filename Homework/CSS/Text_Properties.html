Text Properties - Example

1. การจัดแนวข้อความ (text-align)

ข้อความนี้จัดชิดซ้าย

ข้อความนี้จัดกึ่งกลาง

ข้อความนี้จัดชิดขวา

ข้อความนี้จัดเต็มบรรทัด (justify) เพื่อให้ชิดซ้ายและขวาโดยอัตโนมัติ

2. การตกแต่งข้อความ (text-decoration)

ข้อความนี้ขีดเส้นใต้ (underline)

ข้อความนี้ขีดฆ่า (line-through)

ข้อความนี้ไม่มีการตกแต่ง (none)

3. การกำหนดระยะบรรทัด (line-height)

ระยะบรรทัด 1.5 เท่า - เหมาะสำหรับการอ่านแบบสบายตาในบทความทั่วไป

ระยะบรรทัด 24px - ควบคุมความสูงบรรทัดแบบคงที่
