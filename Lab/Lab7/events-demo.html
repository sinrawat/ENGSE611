JavaScript Events Demo

ตัวอย่าง Event ยอดนิยม 7 แบบ พร้อมแสดงผลในกล่อง Log ด้านล่าง

1) Mouse Event – onclick

เมื่อคลิกปุ่ม จะมีข้อความแสดงใน Log

2) Keyboard Event – onkeyup

ลองกดปุ่มบนคีย์บอร์ดใด ๆ แล้วปล่อย (โฟกัสอยู่ที่หน้าเว็บ)

ระบบจะบอกว่า “คุณปล่อยปุ่มอะไร” ใน Log

3) Input Event – oninput

ขณะพิมพ์ ระบบจะแสดงข้อความที่กำลังพิมพ์ใน Log แบบ real-time

4) Window Event – onresize

ลองย่อ/ขยายหน้าต่างเบราว์เซอร์ จะเห็นขนาดใหม่ใน Log

5) Touch Event – ontouchstart

แตะบนกล่องนี้ (บนมือถือ/แท็บเล็ต)

บน Desktop ส่วนใหญ่จะไม่ยิง touch event, ใช้ทดสอบบนมือถือ

6) Clipboard Event – oncopy

เลือกข้อความนี้แล้วกด Ctrl+C / Copy:

ข้อความตัวอย่างสำหรับทดสอบ event oncopy

7) Drag & Drop Events – ondragstart + ondrop

ลากฉันไปวางด้านขวา
Drop Zone

ลากกล่องสีส้มไปปล่อยบนพื้นที่ Drop Zone แล้วดู Log
